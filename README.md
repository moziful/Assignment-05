1. Var is old system to declare variable. It is a function scoped variable. On the other hand, const and let is modern variable declaration method. If the value needs to be changed, we use let. Otherwise we should use const. Var may lead to bugs due to its hoisting behavior.

2. The spread operator gives us simple and important way to expand elements from arrays, strings, or objects. We can get all element from those arrays, strings or objects just by using spread operator very easily and quickly.

3. map(): Create a new array of the same length, items arer modified by our function.
filter(): Creates a new array containing only the items fulfill specific condition provided.
forEach(): It is a loop. It executes the function for all item but returns nothing.

4. Arrow Function is a shortcut way to create function just by using this (=>) simble as arrow. Its structure looks like:
const functionName = (Parameters) => {  // Execute function codes };

5. Template literals is a modern way to work with string. We can use string anywhere in the html through it. We can use multiline strings too. It is enclosed by backticks (`) instead of quotation marks (" or ').
